## 1. General Debate Workflow: 

When : session begins or returning from a caucus
Any motions on the floor ? 
if motions : 
    add motions to list and decide the one to discuss 
    -> go to [motion workflow (2) (3) (4) (5)]
else : 
    -> default: go to [2. General Speakers List Workflow]

## 2. General Speakers List Workflow 

Open speaking queue 
if delegates register: 
    chair selects delegates and builds queue
    for every delegate in the queue: 
        -> go to [3.1 Individual Speech Workflow, general speech type]
else: 
    -> go to [1.General Debate Workflow]

## 3. Individual Speech Workflow

Chair calls delegate for speech 
Delegate speaks for time <= max time 
if time left: 
    open questions 
    chair selects delegates for questions
    delegate asks questions 
    speaker responds 
    end -> back to current workflow
else: 
    -> back to current workflow

## 4. Moderated Caucus Workflow 

Present parameters: topic, total time, speaking time...
Chair ask for voluntary speakers 
Chair builds queue 
for every delegate in the queue: 
    -> go through [3.2 Individual Speech Workflow, mod caucus speech type]
    end 
-> go to [1.General Debate Workflow]

## 5. Unmoderated Caucus Workflow

start caucus time 
form delegation blocs 
discuss within blocs 
for each bloc: 
    negociate 
    write working paper
    possibly redact draft resolution 
    submit working paper
end 
-> go to [1.General Debate Workflow]
